# DKMK2 



![DKMK2](dkmk22.png){ width="500" }



## Quick facts



- **Developed by:** *NSL*

- **Release:** *November 2020*

- **Origin:** *China*

- **Status:** *Discontinued*

- **Production:** *Batch*

- **Scale:** *1/24-1/28*

- **Body mounting:** *Magnet mounting / Mini-Z*

- **Materials:** *Carbon fiber, injection moulded plastic, brass*
